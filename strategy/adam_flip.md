# 翻亞當（第二映像圖）演算法與策略

來源：Jack Li 新手練習系列 — 亞當理論實戰應用  
用途：**評估走勢滿足距離（出場目標區）與走勢可能結束的位置**

> 翻亞當是「交易計畫工具」，不是進場依據。進場仍需等破底翻訊號。

---

## 核心概念

市場在完成一段移動後，傾向以**等距方式**延伸下一段走勢。翻亞當透過量測「翻立前的基準距離」，來推算「翻立後的滿足目標」。

```
翻立訊號（Adam Flip）→ 量測基準距離 → 從翻立點投影等距 → 得到滿足區
```

---

## 演算法：三步驟

### Step 1：確認翻立訊號（Flip Detection）

只有 K 棒**實體**明確突破關鍵位，才觸發翻亞當。影線突破（只有影線破，收回）= 不翻。

| 方向 | 觸發條件（任一）|
|------|----------------|
| 多方翻立 | 長紅K 實體 / 收盤創新高 / 實體突破盤整高點 |
| 空方翻立 | 長黑K 實體 / 收盤創新低 / 實體跌破盤整低點 |

**連續噴出的處理：**
找「第一根明確離開盤整區」的那根 K 棒翻。後續連續大K 若左側沒有新的壓力/支撐點，不需重翻；若突破左側另一個高/低點，視為新的翻立。

---

### Step 2：選取量測基準（Measurement Selection）

根據盤型選擇以下三種模式之一：

---

#### 模式 A：盤整翻盤整（最常用）

**觸發情境**：翻立訊號是由盤整區間被突破引起。

```
量測：盤整高點 - 盤整低點 = 基準距離 X
投影：從翻立點（突破邊界）往突破方向投影 X

空方範例：
  盤整高點 = 100
  盤整低點 = 90
  X = 10
  跌破低點（翻立點 = 90）
  目標 = 90 - 10 = 80
```

---

#### 模式 B：N 字型轉折

**觸發情境**：行情在趨勢中形成一個 N 字型轉折後繼續原方向，或從趨勢中的擺動高/低點轉向。

```
量測：前一段擺動的 A→B 距離
投影：從 B 點投影相同距離

空方範例（趨勢由多轉空）：
  A = 前波高點 100（swing high）
  B = 回測低點 90（swing low = 翻立點）
  AB = 10
  目標 = B - AB = 90 - 10 = 80

多方範例（趨勢由空轉多）：
  A = 前波低點 80（swing low）
  B = 反彈高點 90（swing high = 翻立點）
  AB = 10
  目標 = B + AB = 90 + 10 = 100
```

---

#### 模式 C：缺口或大 K 棒（特殊處理）

**觸發情境**：翻立K 棒是缺口跳開，或實體遠大於近期平均（≥3× 均值）。

規則：**實體對實體配對**（不用影線），以翻立K 實體的邊界做基準重新量起。

```
缺口K 範例（空方）：
  翻立K：開 = 95，收 = 88（實體大小 = 7）
  基準 = 實體大小 7
  目標 = 88 - 7 = 81
```

---

### Step 3：計算滿足區（Target Zone）

```
空方目標 = 翻立點 - 基準距離
多方目標 = 翻立點 + 基準距離

滿足區 = 目標 ± 0.3%（緩衝帶）
```

---

## 演算法流程圖

```
開始
│
├─ 是否有K棒實體明確突破關鍵位？
│   ├─ 否 → 不翻亞當，等待
│   └─ 是 → 繼續
│
├─ 翻立前是否有明確盤整結構？
│   ├─ 是 → 模式 A（盤整翻盤整）
│   └─ 否 → 繼續
│
├─ 是否有 N 字型擺動高/低點？
│   ├─ 是 → 模式 B（N字型轉折）
│   └─ 否 → 繼續
│
├─ 翻立K 是缺口或超大K？
│   └─ 是 → 模式 C（實體對實體）
│
├─ 計算目標 = 翻立點 ± 基準距離
│
├─ 設滿足區 = 目標 ± 0.3%
│
└─ 驗證：是否與左側重要支撐/壓力重合？
    ├─ 重合 1 個 → 信心中等
    ├─ 重合 2+ 個 → 信心高，高機率反轉區
    └─ 無重合 → 注意，可能超跌/超漲
```

---

## 多時框疊加（Multi-Timeframe Stack）

分別在不同時框計算翻亞當目標，用來區分**短線出場點**和**趨勢結束點**。

| 時框 | 用途 |
|------|------|
| 5m / 15m 亞當目標 | 短線出場，第一次獲利了結（出 1/2 倉） |
| 60m 亞當目標 | 波段出場，主要滿足區 |
| 日K 亞當目標 | 趨勢終點，大波段完整目標 |

**重合判斷（±0.5% 以內視為重合）：**
- 小時框目標 ≈ 大時框目標 → **強力滿足區**，容易亂掃洗盤
- 到達此區：縮停損 / 出 1/2 部位 / 準備反轉進場（破底翻）
- 不重合 → 小時框先出一半，大時框再全出

## 出場策略（Exit Strategy）

### 標準出場流程

```
1. 盤前標記：翻亞當算出滿足區，標在圖上
2. 持有期間：不因途中波動提前出場
3. 進入滿足區（±0.3%）：
   ├─ 若只有小時框目標 → 出 1/2，帶剩餘單等大目標
   ├─ 若大小時框重合 → 全出，防止亂掃
   └─ 若滿足區 + 左側重要S/R → 信心高，可全出
4. 超漲/超跌超過目標：不追，等下一個亞當翻立再評估
```

### 做多的出場應用（多方行情中）

```
多方翻亞當：算出多方滿足區（向上目標）

進入多方目標區時：
- 觀察是否出現空方亞當翻立訊號（長黑K / 創新高後反跌 / 跌破盤整）
- 有翻立訊號 → 出場（或做空）
- 無翻立訊號 → 持有，等下一根確認

大小亞當目標重合時：
- 不等訊號，直接先出一半
```

### 做空的出場應用（空方行情中）

```
空方翻亞當：算出空方滿足區（向下目標）

進入空方目標區時（= 下跌滿足，可能反轉向上）：
- 觀察是否有多方亞當翻立訊號（破底翻）
- 有 → 空方出場，換方向做多（此為「空方滿足區做多」例外條件）
- 無 → 持有，等確認
```

## 特殊情況處理

### 盤整區間內
**不翻亞當。** 打開盤面先畫支撐/壓力兩條線，盤整中怎麼翻目標都在區間內，無意義。等突破後再翻。

### 破底翻時不直接翻多方亞當
做破底翻進場時，底部只是「潛在反轉點」，尚未確認趨勢改變。**必須等價格實體突破上緣壓力線（多方亞當翻立成立）後**，才能翻多方亞當算目標。

### 行情走不到或超過目標
亞當理論不是精確科學。市場可能：
- 提前反轉（未到目標就翻）→ 看翻立訊號，有訊號就出
- 超過目標繼續走 → 等下一個轉折點重新翻亞當
- 到達目標亂掃 → 大小亞當重合區最常見，提前縮停損

---

## 信心評分（Confidence Score）

到達滿足區時，以下每一項 +1 分，最高 3 分：

| 加分項目 | 說明 |
|---------|------|
| 左側重要壓力/支撐重合 | 歷史高低點、盤整邊界在附近 |
| 大小亞當目標重合 | 多時框滿足區疊加 |
| 均線（20/60/240MA）在附近 | 動態支撐壓力確認 |

- 0 分：謹慎，滿足區可能失效
- 1 分：中等信心，出一半
- 2-3 分：高信心，可全出 / 準備反向進場

## 記憶口訣

> **翻立看實體，量測選模式，目標投等距，滿足區全出**

- 模式 A：盤整高度往下投
- 模式 B：N字AB距離往下投
- 模式 C：缺口K實體往下投
- 大小重合 = 亂掃區，先跑
