判斷偏多（符合 2 條以上才算）：

- 日K 收盤在 20MA 之上
- 昨天不是長黑K 且收盤接近低點

### 結果處理
| 判斷 | 行動 |
|------|------|
| 偏多 ✅ | 繼續第二關 |
| 盤整 | 縮口數，謹慎進場 |
| 偏空（多方必死盤）❌ | **MNQ 全天不打多，觀望或找空方機會** |

### ⚡ 例外：空方滿足區 + 60K 支撐 → 可做多
即使日K 偏空，同時滿足以下兩條件時，可考慮小口做多：

| 條件 | 說明 |
|------|------|
| ✅ 價格到達**空方滿足區** | 用「翻亞當（第二映像圖）」計算的下跌目標區 |
| ✅ **60K 有明確支撐** | 水平支撐 / 60K 前低 / 亞當底區域站穩 |

**操作規則（例外模式）：**
- 口數縮半，最多 1 口，不加碼
- 進場仍需等 5m/15m 破底翻訊號（翻亞當只是計畫，不是進場依據）
- 止損設在滿足區下方（破了就走，不磨）
- 目標只到下一個壓力，不貪拉
- 兩條不同時成立 → 回歸「偏空不打多」

> 邏輯：翻亞當算出空方能量耗盡的目標位，60K 支撐確認有止跌結構，到達時等破底翻訊號進多。

### 翻亞當（第二映像圖）— 計算滿足區

**核心概念**（來源：Jack Li 亞當理論新手練習系列）
> 翻亞當用於**判斷方向與滿足距離**，不能作為進場依據。賺計畫中的錢，不追漲殺跌。

**觸發條件（三擇一，K棒實體必須明確突破）：**

| 情境 | 計算方式 |
|------|------|
| 盤整被跌破 | 量盤整區高度，從跌破點往下等距投影（盤整翻盤整） |
| 趨勢中的 N 字轉折 | 量前一段反彈（A→B 高低點距離），從 B 點往下投影同距離 |
| 缺口或巨大 K 棒 | 實體對實體配對（不用影線），重新量起 |

**注意事項：**
- 只有影線破（長上影線/下影線後縮回）= 突破失敗，**不翻亞當**
- 連續噴出時：找**第一根離開盤整**的那根來翻，後續大K若沒突破新的高/低點不需重翻
- 做破底翻時：**不能在底部直接翻多方亞當**，必須等價格明確突破上緣壓力後才翻

**滿足區到達時的注意：**
- 大格局亞當 + 小格局亞當滿足區重合 → 容易「亂掃」洗盤，提高警覺，**考慮先出場**
- 滿足區若與左側重要支撐/壓力或**開盤價**重合 → 兩訊號疊加，反轉機率更高
- 市場可能走不到或超跌超漲 → 到達區間就觀察，**不押注一定反轉**

### Al Brooks 補充
開盤後前 20 分鐘，看前 3-5 根 15 分K：
- 熊K 連續、回調都是弱牛K → Always In Short → 今天不打多
- 出現 Spike and Channel（大跌棒後跟通道）= 趨勢日，不做反轉

## 第二關：60K 多方亞當翻立（開盤前確認）

**目的**：確認 60 分K 上已有多方方向，不是在空頭或未翻立的結構中打多

**理論依據**：亞當理論（J. Welles Wilder）— 不預測，只跟隨市場已發生的事實

### 多方亞當翻立條件（任一即可）
| 訊號 | 說明 |
|------|------|
| ✅ 60K 出現**長紅K** | 多方力道強烈介入 |
| ✅ 60K **創新高** | 趨勢序列確認向上 |
| ✅ 60K **突破盤整高點** | 多方突破壓力，方向明確 |

### 空方亞當翻立（遇到這些 → 不打多）
| 訊號 | 說明 |
|------|------|
| ❌ 60K 出現長黑K | 空方力道介入 |
| ❌ 60K 創新低 | 趨勢向下確認 |
| ❌ 60K 跌破盤整低點 | 空方突破，方向向下 |

### 結果處理
| 判斷 | 行動 |
|------|------|
| 多方亞當翻立 ✅ | 繼續第三關 |
| 尚未翻立（盤整） | 縮半口，等翻立訊號出現 |
| 空方亞當翻立 ❌ | **不打多，觀望** |

### 翻立後的兩種進場模式
1. **有回測支撐** → 等守住後用**破底翻**進場
2. **直接噴出沒有回測** → 用**突破回踩**進場（突破後第一次拉回）

> 6/24 自診：「基本上今天的商品都沒有60K多方亞當撐腰」
> → 60K 未翻立就進場，情緒單連發，本金燒完，錯過後來 GC 的真正翻轉

> 6/30 教訓：60K 多方亞當已翻立（突破盤整高點），但沒有給破底翻機會
> → 應改用突破回踩模式，而非繼續等一個不會來的下殺

## 開盤價作為日內重要撐壓

每日開盤第一根 K 棒的**開盤價**在當天盤中具有撐壓意義：

| 價格位置 | 意義 | 操作含義 |
|----------|------|----------|
| 價格從上方**回踩開盤價** | 開盤價當**支撐** | 可在此區找破底翻進場（需搭配互換訊號）|
| 價格從下方**反彈撞開盤價** | 開盤價當**壓力** | 做多目標不貪過開盤價；空方反彈賣點 |

### ⭐ 高機率特殊場景：前一日出現下尖尖

**前一日日K 出現下尖尖（長下影線 / 亞當底 / 錘頭）→ 隔天開盤價是重要支撐**

```
昨日：跌破低點 → 長下影線 → 收回（下尖尖）
今日：開盤價 = 昨日收盤區域 = 重要撐壓
當價格回踩開盤價 → 一定要找機會做多（強制尋找進場）
```

**邏輯：** 下尖尖代表昨天空方嘗試下殺被多方強力收回，這個守住的位置（今日開盤附近）就是多方願意大力承接的區域，回踩時再次出現此結構的勝率高。

**操作規則：**
- 前一日有下尖尖 → 今日開盤價**升格為主要支撐**（不只是輔助）
- 價格回踩開盤價 → **主動尋找破底翻或互換進場**，不能錯過
- 止損設在下尖尖低點以下（昨日下影線最低點）
- 仍需第二關（60K 多方亞當）確認，但信心可以加大

**使用注意：**
- 開盤價是**輔助撐壓**，不是獨立進場依據，仍需三關通過
- 與 60K 結構支撐、翻亞當滿足區重合時，訊號強度倍增
- 價格在**開盤價之下整理**（開盤後始終低於開盤價）→ 偏空訊號，多方謹慎
- 美股 MNQ 通常以 **09:30 ET** 的開盤價為準

> Al Brooks 補充：開盤後前 30 分鐘若收盤價持續在開盤價以下 → Always In Short → 今天不打多

## 第三關：當下盤口（進場前 30 秒）

**目的**：確認進場訊號是計畫內的，不是情緒驅動的

同時滿足才按：
- ✅ 在 60K 計畫的支撐區、亞當底、或**開盤價**附近（不是隨便一個地方的破翻）
- ✅ 若第一次破底翻在支撐區被收回後又跌回（= 破底翻失敗），**不追第二次衝動單**，改等「W 底第二低站回支撐」再進（回測統計：5m/15m 上約 59% 反轉需二次測試，等第二低的假陽性率僅約 8–14%）
- ✅ 5m/15m 有互換訊號
- ✅ 不是剛被止損後的第一個衝動

### 摸單口數規則
- **摸單 = 尚未確認的試探進場**（支撐到位但訊號尚未完全成立）
- 摸單口數 = 正常口數的一半（例：正常 2 口 → 摸單 1 口）
- 確認站穩後，才加到正常口數

### 情緒單防護規則
- 被止損後 → **強制等 15 分鐘**再看圖
- 問自己：「這筆進場是在盤前計畫的支撐區嗎？」
- 如果不是 → 這筆是情緒單，不進

### 🟢 手上有子彈時的彈性規則（當日淨損益為正）

當日已有獲利（淨損益 > 0）且三關通過，可啟用以下彈性：

| 彈性 | 條件 | 限制 |
|------|------|------|
| **加碼** | 第一筆已在獲利中，出現新的計畫支撐區訊號 | 加碼口數 ≤ 原始口數，止損設在加碼點下方（獨立於原單） |
| **試單次數提升至 3 次** | 同一支撐區反覆測試（符合 W 底或複雜底結構） | 每次試單口數縮小（例：正常 2 口 → 試單改 1 口），總虧損上限不變 |

**使用前提**：
- 「有子彈」= 當日實現損益 > 0，不是「帳面獲利」
- 加碼的止損被點 → 仍觸發強制收工（不因有子彈而豁免）
- 3 次試單均失敗 → 當下支撐判斷有誤，今天放棄該區域

> 7/8 教訓：晚盤 28,996 → 29,002 → 28,993 三次進場，前兩次失敗是 W 底試底結構，
> 有子彈時可以小口試（1 口），第三次確認站穩後再加到正常口數。

### 🔴 每日強制收工條件（硬規則，無例外）
觸發任一條件 → **立刻關閉下單視窗，今天不再進場**：

| 條件 | 說明 |
|------|------|
| 加碼單被止損 | 加碼被點 = 方向錯，此時心態最危險（即使有子彈也收工）|
| 連續虧損 3 筆 | 無論原因，連 3 停，今天收工 |
| 早盤獲利被吃光 | 淨損益回歸 0 → 停止，保住今天不虧 |
| 同一方向連敗 2 筆 | 市場在告訴你方向錯了（有子彈彈性：可用 3 次試單替代，但每次縮口）|
| 當日虧損 ≥ $650 | 心理失控臨界點（自觀察回測），超過後翻本心態出現，強制收工 |

**日內動態口數規則（根據虧損狀態）：**

| 當日虧損 | 口數管理 |
|---------|---------|
| < $325 | 正常或可放大（夜盤彈性適用） |
| $325 ~ $649 | 回到基本口數，不加碼 |
| ≥ $650 | 強制收工 |

> $650 = 根據自觀察回測確立的心理失控臨界點，不是隨機數字。$325（50%）為警戒線。

> 7/2 教訓：12:38 加碼單被點（-$58），沒有停下來 →
> 13:42 起連續 13 筆全輸 -$1,136，W 底在 29,860 沒吃到。
> **加碼被點那一刻如果收工，今天是 +$214，不是 -$864。**

## 記憶口訣

> **日線亞當翻多 → 60K 亞當翻立 → 小格局互換或回踩，三關全過才按**

失敗案例：
- 6/23：第一關失敗（多方必死盤仍打多）→ -$860
- 6/24：第二關失敗（60K 未翻立）+ 第三關失敗（情緒單）→ -$800

成功案例：
- 6/26：三關全過 → 60K 翻立後尖尖旁邊進場 → +$1,070

錯過案例（無失誤，只是模式不同）：
- 6/30：60K 多方亞當翻立（突破盤整高點）→ 無下殺無破底翻機會 → 應改用突破回踩模式
