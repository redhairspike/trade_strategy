## 核心策略：破底翻（False Breakdown Reversal）

### 定義
價格跌破支撐 → 引發止損獵殺 → 快速收回支撐之上 → 做多

### 盤前規劃 SOP（開盤前依序完成）
```
Step 1：切換 60K → 劃出撐壓（前高/前低/水平支撐/翻亞當滿足區）
Step 2：觀察 60K 亞當方向
         多方翻立 ✅ / 未翻立 ⚠️ / 空方翻立（記錄次數）❌
Step 3：確認今日開盤價位置
         開在支撐之上？撐壓之間？壓力之下？
         前一日下尖尖？→ 今日開盤價升格主要支撐
Step 4：綜合判斷
         🟢 強偏多 → 積極做多
         🟡 謹慎   → 縮口試水
         🔴 必死盤 → 今天不打多
```

### 三層執行框架
```
第一層：日線趨勢濾網（開盤前完成）
    ↓ 通過才進入
第二層：60K 邊界識別（開盤前確認）
    ↓ 有亞當底撐腰才進入
第三層：5m/15m 進場訊號（進場前 30 秒）
    ↓ 互換訊號 + 在計畫支撐區 = 進場
```

## 趨勢濾網（三關制）

### 第一關：日線方向（開盤前完成）
判斷是否偏多（符合 2 條以上）：
- 最近 3 根日K：高低點序列沒有下跌（不連跌）
- 日K 收盤在 20MA 之上
- 昨天不是長黑K 且收盤接近低點

→ **偏空（多方必死盤）= MNQ 不打多方破底翻，全天觀望**

**例外：空方滿足區 + 60K 支撐 → 可小口做多**
- 條件 1：價格到達空方滿足區（AB=CD / 重要結構支撐）
- 條件 2：60K 有明確支撐（水平撐 / 前低 / 亞當底）
- 兩條同時成立才用，口數縮半，止損在滿足區下方

### 第二關：60K 多方亞當翻立（開盤前確認）
**亞當理論（J. Welles Wilder）**：不預測，只跟隨市場已發生的事實。

多方亞當翻立條件（任一即可）：
- 60K 出現**長紅K**
- 60K **創新高**
- 60K **突破盤整高點**

任一條件成立 → 60K 多方亞當翻立 → 趨勢向上 → 可做多

空方亞當翻立條件（任一即可）：
- 60K 出現長黑K
- 60K 創新低
- 60K 跌破盤整低點

→ **60K 多方亞當未翻立 = 縮口數或不進**

### 第三關：當下盤口（進場前）
同時滿足才進：
- 在 60K 計畫的支撐區、亞當底、或**開盤價**附近
- 5m/15m 有互換訊號
- 不是剛被止損後的第一個衝動（強制等一根 K 再看）

**注意：** 多方亞當翻立後有兩種進場模式：
1. 價格回測支撐 → **破底翻**（有下殺才用）
2. 價格沒有回來 → **突破回踩**（盤整後直接噴的日子用）

→ **被點後馬上想再進 = 情緒單，等 15 分鐘**

## 進場確認規則
- 破底後必須在 **2-3 根 K 棒內**收回支撐之上
- 破底K 成交量高，回收K 成交量縮小 = 良好訊號
- **長黑K 隔壁不衝動進場**，等下一根確認
- 互換訊號（均線交叉）在 60K 支撐區附近才有效
- **摸單口數減半**：摸單（試探支撐，尚未確認）= 正常口數的一半（例：正常 2 口 → 摸單 1 口）

## 開盤價撐壓規則（日內）
- 價格從上方**回踩開盤價** → 開盤價視為**支撐**，可找破底翻
- 價格從下方**反彈撞開盤價** → 開盤價視為**壓力**，多單目標不貪過此位
- 開盤後 30 分鐘收盤價持續低於開盤價 → 偏空，今天不打多方
- 開盤價 + 60K 支撐重合 → 訊號強度倍增

**⭐ 特殊場景：前一日出現下尖尖（長下影線 / 亞當底）**
- 隔天開盤價**升格為主要支撐**（強制尋找機會做多，不能錯過）
- 止損設在昨日下尖尖低點以下
- 邏輯：下尖尖 = 空方試圖下殺被多方強力收回，這個守住位置是高機率多方進場區

## 停損 / 停止規則
- 同一商品連續 **3 次** 破底翻失敗 → 當天停止交易該商品
- 本金不足時降低口數（例如本金剩 $500 → 改打 2 口）
- 停損設定不能太大，必要時用口數控制風險而非擴大停損距離

## 日內虧損控制（心理臨界點規則）

**$650 = 根據自觀察回測的心理失控臨界點**（超過後後續交易容易失控、翻本心態出現）

| 當日虧損 | 狀態 | 操作 |
|---------|------|------|
| < $325 | 正常範圍 | 可維持或放大口數（夜盤彈性適用） |
| $325 ~ $649 | ⚠️ 警戒 | 回到基本口數，謹慎模式，不加碼 |
| ≥ $650 | 🔴 觸發臨界 | **強制收工，今天不再進場** |

> 夜盤口數彈性邏輯：夜盤進場時若當日虧損 < $325，允許放大口數；此為有依據的動態管理，不是情緒加倉。

## 有子彈（當日淨損益 > 0）時的彈性規則
- **加碼**：第一筆獲利中 + 新的計畫支撐訊號 → 可加碼，口數 ≤ 原始口數
- **試單次數提升至 3 次**：同一支撐區 W 底結構 → 試單改 1 口，第三次確認後才正常口數
- 加碼被點 → 仍觸發強制收工，不因有子彈豁免
- 3 次試單全敗 → 放棄該區域，今天不再進

## 出場規則

- 不要太早出場（主要失誤之一）
- 有計畫出場點就照計畫，不因當下走勢改變

## 價格行為學補充（Al Brooks 概念）

- **Always In 方向**：開盤前 20 分鐘 K 棒的方向告訴你今天做多還是做空
- **Spike and Channel**：開盤大趨勢K（spike）後跟通道 = 趨勢日，不做反轉
- **熊K 強度判斷**：大實體、收盤接近低點、上影線短 = 空方控制中
- **反轉失敗次數**：趨勢日早期反轉嘗試失敗，通常第 2-3 次才成功

## 亞當理論（Adam Theory）— J. Welles Wilder

核心精神：**不預測，只跟隨市場已發生的事實**

### 多方亞當翻立（任一條件即可）
| 訊號 | 含義 |
|------|------|
| 長紅K 出現 | 多方力道強烈介入 |
| 創新高 | 趨勢序列確認向上 |
| 突破盤整高點 | 多方突破壓力，方向明確 |

### 空方亞當翻立（任一條件即可）
| 訊號 | 含義 |
|------|------|
| 長黑K 出現 | 空方力道強烈介入 |
| 創新低 | 趨勢序列確認向下 |
| 跌破盤整低點 | 空方突破支撐，方向向下 |

### 應用層次
- **日K 亞當** → 決定今天整體偏多或偏空（第一關）
- **60K 亞當** → 決定這段行情是否有結構支撐（第二關）
- **5m/15m 亞當** → 進場訊號確認（第三關的互換）

### 翻立後的兩種進場模式
1. **有回測** → 等支撐守住後破底翻進場
2. **沒有回測（直接噴）** → 突破回踩進場（等突破後第一次拉回）

### 翻亞當（第二映像圖）— 計算滿足區
**用途**：計算行情「方向與滿足距離」，不是進場依據，進場還是靠破底翻訊號。

計算方式（三擇一）：
- **盤整被突破** → 量盤整高度，從突破點等距投影（盤整翻盤整）
- **N字型轉折** → 量 A→B 距離，從 B 點等距投影
- **缺口/大K棒** → 實體對實體配對，重新量起

觸發規則：
- K棒實體必須明確突破，只有影線不算
- 連續噴出：找第一根離開盤整的那根翻
- 做破底翻時：等價格明確突破上緣壓力後再翻多方亞當

滿足區到達時：
- 大小亞當滿足區重合 → 容易亂掃，考慮先出場
- 滿足區 + 左側重要支撐重合 → 兩訊號疊加，反轉機率更高
- **偏空日** + 到達**空方滿足區** + **60K 支撐** → 例外可小口做多（縮半口，破了就走）

## Claude 使用提示

當 Spike 要求分析交易時：
1. 優先看進場是否通過三關濾網
2. 檢查是否在計畫支撐區進場（非情緒單）
3. 對照盤前規劃與實際執行的差異
4. 指出情緒單（被點後馬上再進）
5. 不要只講策略，要對照當天日誌的具體情境

當 Spike 分享 Notion 日誌 URL 時：
- 使用 Notion MCP 讀取頁面內容
- 從圖片**檔名**判斷進場品質（含「情緒」「進場不好」「太早」等關鍵字）
- 交叉對比盤前規劃 vs 盤後檢討 vs 進場圖

當 Spike 問趨勢判斷時：
- 先問當天日K 狀態
- 用 Al Brooks Always In 概念協助判斷
- 提醒三關濾網的第一關
